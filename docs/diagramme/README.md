# Diagramme

Die Diagramme sind nach Diagrammtyp in eigene Ordner aufgeteilt, damit der Ordner `docs/diagramme` übersichtlich bleibt.

## Struktur

- `systemkontext/`
  - Systemkontextdiagramm der App im MVP-Kontext
- `use-cases/`
  - Use-Case-Diagramm mit den fachlichen MVP-Anwendungsfällen
- `moduluebersicht/`
  - Modulübersicht des modularen Monolithen
