# Diagramme

Die Diagramme sind nach Diagrammtyp in eigene Ordner aufgeteilt, damit der Ordner `docs/diagramme` uebersichtlich bleibt.

## Struktur

- `systemkontext/`
  - Systemkontextdiagramm der App im MVP-Kontext
- `use-cases/`
  - Use-Case-Diagramm mit den fachlichen MVP-Anwendungsfaellen
- `moduluebersicht/`
  - Moduluebersicht des modularen Monolithen
- `Klassendiagramm/`
  - fachliches Datenmodell des MVP
- `Aktivitätsdiagramme/`
  - Aktivitaetsdiagramme fuer die Kern-Workflows Zeitlimit, Schlafenszeit und Bewegungspausen
